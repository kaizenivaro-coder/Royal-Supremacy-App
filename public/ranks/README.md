# MLBB Rank Icons

Transparent 400x400 PNG rank emblems used by the Royal Supremacy profile UI.

Included ranks:

- Unranked
- Grandmaster
- Epic
- Legend
- Mythic
- Mythical Honor
- Mythical Glory
- Mythical Immortal

Source reference:

- https://www.novazenn.com/2024/01/icon-logo-rank-ml-png.html

The source page notes that the displayed Mobile Legends rank images belong to Moonton.
`unranked.png` is a custom Royal Supremacy asset generated for this MVP and is
not an official Mobile Legends/Moonton rank emblem.
