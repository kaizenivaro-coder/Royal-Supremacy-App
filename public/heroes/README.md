# MLBB Hero Icons

This folder contains normalized PNG filenames for the app's `HeroIcon`
component.

The naming rule is:

1. Lowercase the hero name.
2. Replace every group of non-alphanumeric characters with `_`.
3. Trim leading and trailing `_`.

Examples:

- `Chou` -> `chou.png`
- `Yi Sun-shin` -> `yi_sun_shin.png`
- `Yu Zhong` -> `yu_zhong.png`
- `X.Borg` -> `x_borg.png`

These assets were copied from the local `MLBB Default Hero Icons` bundle.
